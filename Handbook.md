The Laws of Chess have two parts: 1. Basic Rules of Play and 2. Competition Rules.   <br/>The English text is the authentic version of the Laws of Chess (which were adopted at the 87th FIDE Congress at Baku, Azerbaijan and 2017 FIDE Presidential Board in Athens) coming into force on 1 July 2017.<br/><br/>In these Laws the words &lsquo;he&rsquo;, &lsquo;him&rsquo;, and &lsquo;his&rsquo; shall be considered toinclude &lsquo;she&rsquo; and &lsquo;her&rsquo;.<br/><br/> <br/><br/>**PREFACE**<br/><br/>The Laws of Chess cannot cover all possible situations that may arise during a game, nor can they regulate all administrative questions. Where cases are not precisely regulated by an Article of the Laws, it should be possible to reach a correct decision by studying analogous situations which are regulated in the Laws. The Laws assume that arbiters have the necessary competence, sound judgement and absolute objectivity. Too detailed a rule might deprive the arbiter of his freedom of judgement and thus prevent him from finding a solution to a problem dictated by fairness, logic and special factors. FIDE appeals to all chess players and federations to accept this view. A necessary condition for a game to be rated by FIDE is that it shall be played according to the FIDE Laws of Chess. It is recommended that competitive games not rated by FIDE be played according to the FIDE Laws of Chess.<br/><br/>Member federations may ask FIDE to give a ruling on matters relating to the Laws of Chess.<br/><br/> <br/><br/>**BASIC RULES OF PLAY**<br/><br/> <br/><br/>**Article 1: The nature and objectives of the game of chess**<br/><br/>1.1<br/><br/>The game of chess is played between two opponents who move their pieces on a square board called a &lsquo;chessboard&rsquo;.<br/><br/>1.2<br/>The player with the light-coloured pieces (White) makes the first move, then the players move alternately, with the player with the dark-coloured pieces (Black) making the next move.<br/><br/>1.3<br/>A player is said to &lsquo;have the move&rsquo; when his opponent&rsquo;s move has been &lsquo;made&rsquo;.<br/><br/>1.4<br/><br/>The objective of each player is to place the opponent&rsquo;s king &lsquo;under attack&rsquo; in such a way that the opponent has no legal move.<br/><br/>1.4.1<br/>The player who achieves this goal is said to have &lsquo;checkmated&rsquo; the opponent&rsquo;s king and to have won the game. Leaving one&rsquo;s own king under attack, exposing one&rsquo;s own king to attack and also &rsquo;capturing&rsquo; the opponent&rsquo;s king is not allowed .<br/><br/>1.4.2<br/><br/>The opponent whose king has been checkmated has lost the game.<br/><br/>1.5<br/><br/>If the position is such that neither player can possibly checkmate the opponent&rsquo;s king, the game is drawn (see Article 5.2.2).<br/><br/> <br/><br/>**Article 2: The initial position of the pieces on the chessboard**<br/><br/>2.1<br/><br/>The chessboard is composed of an 8 x 8 grid of 64 equal squares alternately light <br/>    <br/>    <br/>    Any other physical contact with a piece, except for clearly accidental contact, shall be considered to be intent.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.3     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Except as provided in Article 4.2, if the player having the move touches on the chessboard,with the intention of moving or capturing:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.3.1<br/>    <br/>    <br/>    one or more of his own pieces, he must move the first piece touched that can be moved<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.3.2<br/>    <br/>    <br/>    one or more of his opponent&rsquo;s pieces, he must capture the first piece touched that can be captured<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.3.3<br/>    <br/>    <br/>    one or more pieces of each colour, he must capture the first touched opponent&rsquo;s piece with his first touched piece or, if this is illegal, move or capture the first piece touched that can be moved or captured. If it is unclear whether the player&rsquo;s own piece or his opponent&rsquo;s was touched first, the player&rsquo;s own piece shall be considered to have been touched before his opponent&rsquo;s.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.4      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a player having the move:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.4.1<br/>    <br/>    <br/>    touches his king and a rook he must castle on that side if it is legal to do so<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.4.2<br/>    <br/>    <br/>    deliberately touches a rook and then his king he is not allowed to castle on that side on that move and the situation shall be governed by Article 4.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.4.3<br/>    <br/>    <br/>    intending to castle, touches the king and then a rook, but castling with this rook is illegal, the player must make another legal move with his king (which may include castling with the other rook). If the king has no legal move, the player is free to make any legal move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.4.4<br/>    <br/>    <br/>    promotes a pawn, the choice of the piece is finalised when the piece has touched the square of promotion.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.5      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If none of the pieces touched in accordance with Article 4.3 or Article 4.4 can be moved or captured, the player may make any legal move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.6      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The act of promotion may be performed in various ways:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.6.1<br/>    <br/>    <br/>    the pawn does not have to be placed on the square of arrival,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.6.2<br/>    <br/>    <br/>    removing the pawn and putting the new piece on the square of promotion may occur in any order.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.6.3<br/>    <br/>    <br/>    If an opponent&rsquo;s piece stands on the square of promotion, it must be captured.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.7      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    When, as a legal move or part of a legal move, a piece has been released on a square, it cannot be moved to another square on this move. The move is considered to have been made in the case of:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.7.1<br/>    <br/>    <br/>    a capture, when the captured piece has been removed from the chessboard and the player, having placed his own piece on its new square, has released this capturing piece from his hand,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.7.2<br/>    <br/>    <br/>    castling, when the player&apos;s hand has released the rook on the square previously crossed by the king. When the player has released the king from his hand, the move is not yet made, but the player no longer has the right to make any move other than castling on that side, if this is legal. If castling on this side is illegal, the player must make another legal move with his king (which may include castling with the other rook). If the king has no legal move, the player is free to make any legal move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.7.3<br/>    <br/>    <br/>    promotion, when the player&apos;s hand has released the new piece on the square of promotion and the pawn has been removed from the board.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.8      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A player forfeits his right to claim against his opponent&rsquo;s violation of Articles 4.1 &ndash; 4.7 once the player touches a piece with the intention of moving or capturing it.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    4.9 <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a player is unable to move the pieces, an assistant, who shall be acceptable to the arbiter, may be provided by the player to perform this operation.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Article 5: The completion of the game**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    5.1.1      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The game is won by the player who has checkmated his opponent&rsquo;s king. This immediately ends the game, provided that the move producing the checkmate position was in accordance with Article 3 and Articles 4.2 &ndash; 4.7.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    5.1.2<br/>    <br/>    <br/>    The game is won by the player whose opponent declares he resigns. This immediately ends the game.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    5.2.1<br/>    <br/>    <br/>    The game is drawn when the player to move has no legal move and his king is not in check. The game is said to end in &lsquo;stalemate&rsquo;. This immediately ends the game, provided that the move producing the stalemate position was in accordance with Article 3 and Articles 4.2 &ndash; 4.7.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    5.2.2<br/>    <br/>    <br/>    The game is drawn when a position has arisen in which neither player can checkmate the opponent&rsquo;s king with any series of legal moves. The game is said to end in a &lsquo;dead position&rsquo;. This immediately ends the game, provided that the move producing the position was in accordance with Article 3 and Articles 4.2 &ndash; 4.7.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    5.2.3<br/>    <br/>    <br/>    The game is drawn upon agreement between the two players during the game , provided both players have made at least one move. This immediately ends the game.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **COMPETITION RULES**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Article 6: The chessclock**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.1      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    &lsquo;Chessclock&rsquo; means a clock with two time displays, connected to each other in such a way that only one of them can run at one time.<br/>    <br/>    <br/>    <br/>    <br/>    &lsquo;Clock&rsquo; in the Laws of Chess means one of the two time displays.<br/>    <br/>    <br/>    <br/>    <br/>    Each time display has a &lsquo;flag&rsquo;.<br/>    <br/>    <br/>    <br/>    <br/>    &lsquo;Flag-fall&rsquo; means the expiration of the allotted time for a player.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.1      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    During the game each player, having made his move on the chessboard, shall stop his  own clock and start his opponent&rsquo;s clock (that is to say, he shall press his clock). This &ldquo;completes&rdquo; the move. A move is also completed if:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the move ends the game (see Articles 5.1.1, 5.2.1, 5.2.2, 9.6.1 and 9.6.2), or<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.1.2  <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the player has made his next move, when his previous move was not completed.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.2<br/>    <br/>    <br/>    A player must be allowed to stop his clock after making his move, even after the opponent has made his next move. The time between making the move on the chessboard and pressing the clock is regarded as part of the time allotted to the player.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.3<br/>    <br/>    <br/>    A player must press his clock with the same hand with which he made his move. It is forbidden for a player to keep his finger on the clock or to &lsquo;hover&rsquo; over it.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.4<br/>    <br/>    <br/>    The players must handle the chessclock properly. It is forbidden to press it forcibly, to pick it up, to press the clock before moving or to knock it over. Improper clock handling shall be penalised in accordance with Article 12.9.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.5<br/>    <br/>    <br/>    Only the player whose clock is running is allowed to adjust the pieces.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.2.6<br/>    <br/>    <br/>    If a player is unable to use the clock, an assistant, who must be acceptable to the arbiter, may be provided by the player to perform this operation. His clock shall be adjusted by the arbiter in an equitable way. This adjustment of the clock shall not apply to the clock of a player with a disability.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.3.1      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    When using a chessclock, each player must complete a minimum number of moves or all moves in an allotted period of time including any additional amount of time with each move. All these must be specified in advance.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.3.2<br/>    <br/>    <br/>    The time saved by a player during one period is added to his time available for the next period, where applicable.   <br/>     In the time-delay mode both players receive an allotted &lsquo;main thinking time&rsquo;. Each player also receives a &lsquo;fixed extra time&rsquo; with every move. The countdown of the main thinking time only commences after the fixed extra time has expired. Provided the player presses his clock before the expiration of the fixed extra time, the main thinking time does not change, irrespective of the proportion of the fixed extra time used.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.4      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Immediately after a flag falls, the requirements of Article 6.3.1 must be checked.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.5      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Before the start of the game the arbiter shall decide where the chessclock is placed.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.6      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    At the time determined for the start of the game White&rsquo;s clock is started.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.7.1<br/>    <br/>    <br/>    The regulations of an event shall specify a default time in advance. If the default time is not specified, then it is zero. Any player who arrives at the chessboard after the default time shall lose the game unless the arbiter decides otherwise.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.7.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the regulations of an event specify that the default time is not zero and if neither player is present initially, White shall lose all the time that elapses until he arrives, unless the regulations of an event specify or the arbiter decides otherwise.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.8      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A flag is considered to have fallen when the arbiter observes the fact or when either player has made a valid claim to that effect.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.9      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Except where one of Articles 5.1.1, 5.1.2, 5.2.1, 5.2.2, 5.2.3 applies, if a player does not complete the prescribed number of moves in the allotted time, the game is lost by thatplayer. However, the game is drawn if the position is such that the opponent cannot checkmate the player&rsquo;s king by any possible series of legal moves.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.10.1    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Every indication given by the chessclock is considered to be conclusive in the absence of any evident defect. A chessclock with an evident defect shall be replaced by the arbiter, who shall use his best judgement when determining the times to be shown on the replacement chessclock.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.10.2<br/>    <br/>    <br/>    If during a game it is found that the setting of either or both clocks is incorrect, either player or the arbiter shall stop the chessclock immediately. The arbiter shall install the correct setting and adjust the times and move-counter, if necessary. He shall use his best judgement when determining the clock settings.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.11.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the game needs to be interrupted, the arbiter shall stop the chessclock.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.11.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A player may stop the chessclock only in order to seek the arbiter&rsquo;s assistance, for example when promotion has taken place and the piece required is not available.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.11.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The arbiter shall decide when the game restarts.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.11.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a player stops the chessclock in order to seek the arbiter&rsquo;s assistance, the arbiter shall determine whether the player had any valid reason for doing so. If the player had no valid reason for stopping the chessclock, the player shall be penalised in accordance with Article 12.9.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.12.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Screens, monitors, or demonstration boards showing the current position on the chessboard, the moves and the number of moves made/completed, and clocks which also show the number of moves, are allowed in the playing hall.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    6.12.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The player may not make a claim relying only on information shown in this manner.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Article 7: Irregularities**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If an irregularity occurs and the pieces have to be restored to a previous position, the arbiter shall use his best judgement to determine the times to be shown on the chessclock. This includes the right not to change the clock times. He shall also, if necessary, adjust the clock&rsquo;s move-counter.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.2.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If during a game it is found that the initial position of the pieces was incorrect, the game shall be cancelled and a new game shall be played.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.2.2<br/>    <br/>    <br/>    If during a game it is found that the chessboard has been placed contrary to Article 2.1, the game shall continue but the position reached must be transferred to a correctly placed chessboard.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a game has started with colours reversed then, if less than 10 moves have been made by both players, it shall be discontinued and a new game played with the correct colours. After 10 moves or more, the game shall continue .<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.4.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a player displaces one or more pieces, he shall re-establish the correct position in his own time.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.4.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If necessary, either the player or his opponent shall stop the chessclock and ask for the arbiter&rsquo;s assistance.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.4.3     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The arbiter may penalise the player who displaced the pieces.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.5.1     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    An illegal move is completed once the player has pressed his clock. If during a game it is found that an illegal move has been completed, the position immediately before the irregularity shall be reinstated. If the position immediately before the irregularity cannot be determined, the game shall continue from the last identifiable position prior to the irregularity. Articles 4.3 and 4.7 apply to the move replacing the illegal move. The game shall then continue from this reinstated position.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.5.2   <br/>    <br/>    <br/>    If the player has moved a pawn to the furthest distant rank, pressed the clock, but not replaced the pawn with a new piece, the move is illegal. The pawn shall be replaced by a queen of the same colour as the pawn.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.5.3    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    After the action taken under Article 7.5. 1 or 7.5.2, for the first completed illegal move by a player, the arbiter shall give two minutes extra time to his opponent; for the second completed illegal move by the same player the arbiter shall declare the game lost by this player. However, the game is drawn if the position is such that the opponent cannot checkmate the player&rsquo;s king by any possible series of legal moves.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If, during a game it is found that any piece has been displaced from its correct square, the position before the irregularity shall be reinstated. If the position immediately before the irregularity cannot be determined, the game shall continue from the last identifiable position prior to the irregularity. The game shall then continue from this reinstated position.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.7.1     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a player uses two hands to make a single move (in case of castling, capturing or promotion), it shall be considered as an illegal move. <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.7.2 <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    For the first violation of the rule 7.7.1, the arbiter shall give two minutes extra time to his opponent; for the second violation of the rule 7.7.1 by the same player the arbiter shall declare the game lost by this player. However, the game is drawn if the position is such that the opponent cannot checkmate the player&rsquo;s king by any possible series of legal moves.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.8.1.  <br/>    <br/>    <br/>    If the player presses the clock without making a move, it shall be considered as an illegal move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    7.8.2  <br/>    <br/>    <br/>    For the first violation of the rule 7.8.1, the arbiter shall give two minutes extra time to his opponent; for the second violation of the rule 7.8.1 by the same player the arbiter shall declare the game lost by this player. However, the game is drawn if the position is  such that the opponent cannot checkmate the player&rsquo;s king by any possible series of  legal moves.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Article 8: The recording of the moves**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    In the course of play each player is required to record his own moves and those of his opponent in the correct manner, move after move, as clearly and legibly as possible, in the algebraic notation (Appendix C), on the &lsquo;scoresheet&rsquo; prescribed for the competition.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.1.2<br/>    <br/>    <br/>    It is forbidden to write the moves in advance, unless the player is claiming a draw according to Article 9.2, or 9.3 or adjourning a game according to Guidelines I.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.1.3<br/>    <br/>    <br/>    A player may reply to his opponent&rsquo;s move before recording it, if he so wishes. He must record his previous move before making another.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.1.4  <br/>    <br/>    <br/>    The scoresheet shall be used only for recording the moves, the times of the clocks, offers of a draw, matters relating to a claim and other relevant data.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.1.5 <br/>    <br/>    <br/>    Both players must record the offer of a draw on the scoresheet with a symbol (=). <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.1.6<br/>    <br/>    <br/>    If a player is unable to keep score, an assistant, who must be acceptable to the arbiter, may be provided by the player to write the moves. His clock shall be adjusted by the arbiter in an equitable way. This adjustment of the clock shall not apply to a player with a disability.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The scoresheet shall be visible to the arbiter throughout the game.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The scoresheets are the property of the organiser of the competition.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a player has less than five minutes left on his clock at some stage in a period and does not have additional time of 30 seconds or more added with each move, then for the remainder of the period he is not obliged to meet the requirements of Article 8.1.1.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.5.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If neither player keeps score under Article 8.4, the arbiter or an assistant should try to be present and keep score. In this case, immediately after a flag has fallen the arbiter shall stop the chessclock. Then both players shall update their scoresheets, using the arbiter&rsquo;s or the opponent&rsquo;s scoresheet.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.5.2<br/>    <br/>    <br/>    If only one player has not kept score under Article 8.4, he must, as soon as either flag has fallen, update his scoresheet completely before moving a piece on the chessboard. Provided it is that player&rsquo;s move, he may use his opponent&rsquo;s scoresheet, but must return it before making a move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.5.3<br/>    <br/>    <br/>    If no complete scoresheet is available, the players must reconstruct the game on a second chessboard under the control of the arbiter or an assistant. He shall first record the actual game position, clock times, whose clock was running and the number of moves made/completed, if this information is available, before reconstruction takes place.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the scoresheets cannot be brought up to date showing that a player has overstepped the allotted time, the next move made shall be considered as the first of the following time period, unless there is evidence that more moves have been made or completed.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    8.7<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    At the conclusion of the game both players shall sign both scoresheets, indicating the result of the game. Even if incorrect, this result shall stand, unless the arbiter decides otherwise.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    ** Article 9: The drawn game**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.1.1    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The regulations of an event may specify that players cannot offer or agree to a draw, whether in less than a specified number of moves or at all, without the consent of the arbiter. <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.1.2<br/>    <br/>    <br/>    However, if the regulations of an event allow a draw agreement the following shall  apply:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.1.2.1<br/>    <br/>    <br/>    A player wishing to offer a draw shall do so after having made a move on the chessboard and before pressing his clock. An offer at any other time during play is still valid but Article 11.5 must be considered. No conditions can be attached to the offer. In both cases the offer cannot be withdrawn and remains valid until the opponent accepts it, rejects it orally, rejects it by touching a piece with the intention of moving or capturing it, or the game is concluded in some other way.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.1.2.2<br/>    <br/>    <br/>    The offer of a draw shall be noted by each player on his scoresheet with the symbol (=).<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.1.2.3<br/>    <br/>    <br/>    A claim of a draw under Article 9.2 or 9.3 shall be considered to be an offer of a draw.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.2.1<br/>    <br/>    <br/>    The game is drawn, upon a correct claim by a player having the move, when the same position for at least the third time (not necessarily by a repetition of moves):<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.2.1.1<br/>    <br/>    <br/>    is about to appear, if he first writes his move, which cannot be changed, on his scoresheet and declares to the arbiter his intention to make this move, or<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.2.1.2<br/>    <br/>    <br/>    has just appeared, and the player claiming the draw has the move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.2.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Positions are considered the same if and only if the same player has the move, pieces of the same kind and colour occupy the same squares and the possible moves of all the pieces of both players are the same. Thus positions are not the same if:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.2.2.1<br/>    <br/>    <br/>    at the start of the sequence a pawn could have been captured en passant<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.2.2.2<br/>    <br/>    <br/>    a king had castling rights with a rook that has not been moved, but forfeited these after  moving. The castling rights are lost only after the king or rook is moved.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The game is drawn, upon a correct claim by a player having the move, if:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    he writes his move, which cannot be changed, on his scoresheet and declares to the arbiter his intention to make this move which will result in the last 50 moves by each player having been made without the movement of any pawn and without any capture, or<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.3.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the last 50 moves by each player have been completed without the movement of any pawn and without any capture.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the player touches a piece as in Article 4.3, he loses the right to claim a draw under Article 9.2 or 9.3 on that move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.5.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If a player claims a draw under Article 9.2 or 9.3, he or the arbiter shall stop the chessclock (see Article 6.12.1 or 6.12.2). He is not allowed to withdraw his claim.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.5.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the claim is found to be correct, the game is immediately drawn.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.5.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the claim is found to be incorrect, the arbiter shall add two minutes to the opponent&rsquo;s remaining thinking time. Then the game shall continue. If the claim was based on an intended move, this move must be made in accordance with Articles 3 and 4.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If one or both of the following occur(s) then the game is drawn:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.6.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the same position has appeared, as in 9.2.2 at least five times.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    9.6.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    any series of at least 75 moves have been made by each player without the movement of  any pawn and without any capture. If the last move resulted in checkmate, that shall  take precedence.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Article 10: Points**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    10.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Unless the regulations of an event specify otherwise, a player who wins his game, or wins by forfeit, scores one point (1), a player who loses his game, or forfeits, scores no points (0), and a player who draws his game scores a half point (½).<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    10.2      <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The total score of any game can never exceed the maximum score normally given for  that game. Scores given to an individual player must be those normally associated with the game, for example a score of ¾ - ¼ is not allowed.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Article 11: The conduct of the players**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The players shall take no action that will bring the game of chess into disrepute.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.2.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The &lsquo;playing venue&rsquo; is defined as the &lsquo;playing area&rsquo;, rest rooms, toilets, refreshment area, area set aside for smoking and other places as designated by the arbiter.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.2.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The playing area is defined as the place where the games of a competition are played.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.2.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Only with the permission of the arbiter can:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.2.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    a player leave the playing venue,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.2.3.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the player having the move be allowed to leave the playing area.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.2.3.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    a person who is neither a player nor arbiter be allowed access to the playing area.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.2.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The regulations of an event may specify that the opponent of the player having a move must report to the arbiter when he wishes to leave the playing area.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    During play the players are forbidden to use any notes, sources of information or advice, or analyse any game on another chessboard.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.3.2.1<br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    During a game, a player is forbidden to have any electronic device not specifically approved by the arbiter in the playing venue.<br/>    <br/>    <br/>    <br/>    <br/>    However, the regulations of an event may allow such devices to be stored in a player&rsquo;s bag, provided the device is completely switched off. This bag must be placed as agreed with the arbiter. Both players are forbidden to use this bag without permission of the arbiter.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.3.2.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If it is evident that a player has such a device on their person in the playing venue, the player shall lose the game. The opponent shall win. The regulations of an event may specify a different, less severe, penalty.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.3.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The arbiter may require the player to allow his clothes, bags, other items or body to be inspected, in private. The arbiter or person authorised by the arbiter shall inspect the player, and shall be of the same gender as the player. If a player refuses to cooperate with these obligations, the arbiter shall take measures in accordance with Article 12.9.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.3.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Smoking, including e-cigarettes, is permitted only in the section of the venue designated by the arbiter.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Players who have finished their games shall be considered to be spectators.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    It is forbidden to distract or annoy the opponent in any manner whatsoever. This includes unreasonable claims, unreasonable offers of a draw or the introduction of a source of noise into the playing area.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Infraction of any part of Articles 11.1 &ndash; 11.5 shall lead to penalties in accordance with Article 12.9.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.7<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Persistent refusal by a player to comply with the Laws of Chess shall be penalised by loss of the game. The arbiter shall decide the score of the opponent.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.8<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If both players are found guilty according to Article 11.7, the game shall be declared lost by both players.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.9<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A player shall have the right to request from the arbiter an explanation of particular points in the Laws of Chess.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.10<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Unless the regulations of an event specify otherwise, a player may appeal against any decision of the arbiter, even if the player has signed the scoresheet (see Article 8.7).<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.11<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Both players must assist the arbiter in any situation requiring reconstruction of the game, including draw claims.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    11.12<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Checking three times occurrence of the position or 50 moves claim is a duty of a the players, under supervision of the arbiter.<br/>    <br/>    <br/>    <br/>    <br/> Preface)**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    12.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The arbiter shall see that the Laws of Chess are observed.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/> <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    12.2.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/> <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    12.2.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    take special measures in  <br/>    <br/>    <br/>    12.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The arbiter may appoint assistants to observe games, for example when several <br/>    <br/>    <br/>    <br/>    The arbiter may award either or both players additional time in the event of external disturbance of the game.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    12.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The arbiter must not intervene in a game He shall not indicate the number of moves completed, except in applying Article 8.5 when at least one flag has fallen. opponent,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    12.9.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/> 12.9.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    increasing the points scored in the game by the opponent to the maximum available for that game,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    12.9.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    reducing the points scored in the game by the offending person,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    12.9.6<br/>   <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Appendix A. Rapid chess**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A &lsquo;Rapid chess&rsquo; game is one where either all the moves must be completed in a fixed time of more than 10 minutes but less than 60 minutes for each player; or the time allotted plus 60 times any increment is of more than 10 minutes but less than 60 minutes for each player.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Players do not need to record the moves, but do not lose their rights to claims normally based on a scoresheet. The player can, at any time, ask the arbiter to provide him with a scoresheet, in order to write the moves.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The Competition Rules shall apply if:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.3.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    one arbiter supervises at most three games and<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.3.1.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    each game is recorded by the arbiter or his assistant and, if possible, by electronic means.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.3.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The player may at any time, when it is his move, ask the arbiter or his assistant to show him the scoresheet. This may be requested a maximum of five times in a game. More requests shall be considered as a distraction of the opponent.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Otherwise the following apply:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    From the initial position, once 10 moves have been completed by each player,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    no change can be made to the clock setting, unless the schedule of the event   <br/>     would be adversely affected<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4.1.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    no claim can be made regarding incorrect set-up or orientation of the chessboard.   <br/>     In case of incorrect king placement, castling is not allowed. In case of incorrect   <br/>     rook placement, castling with this rook is not allowed.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the arbiter observes an illegal move has been completed, he shall declare the game lost by the player, provided the opponent has not made his next move. If the arbiter  does not intervene, the opponent is entitled to claim a win, provided the opponent  has not made his next move. However, the game is drawn if the position is such that the opponent cannot checkmate the player&rsquo;s king by any possible series of legal moves. If  the opponent does not claim and the arbiter does not intervene, the illegal move shall  stand and the game shall continue. Once the opponent has made his next move, an  illegal move cannot be corrected unless this is agreed by the players without intervention of the arbiter.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    To claim a win on time, the claimant may stop the chessclock and notify the arbiter. For the claim to be successful, the claimant must have time remaining on his own clock  after the chessclock has been stopped. However, the game is drawn if the  position is such that the claimant cannot checkmate the player&rsquo;s king by any possible  series of legal moves.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the arbiter observes both kings are in check, or a pawn on the rank furthest from its  starting position, he shall wait until the next move is completed. Then, if an illegal  position is still on the board, he shall declare the game drawn.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.4.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The arbiter can also call a flag fall, if he observes it.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The regulations of an event shall specify whether Article A.3 or Article A.4 shall apply for the entire event.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Appendix B. Blitz**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A &lsquo;blitz&rsquo; game is one where all the moves must be completed in a fixed time of 10 minutes or less for each player; or the allotted time plus 60 times any increment is 10 minutes or less.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The penalties mentioned in Articles 7 and 9 of the Competition Rules shall be one minute instead of two minutes.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The Competition Rules shall apply if:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.3.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    one arbiter supervises one game and<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.3.1.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    each game is recorded by the arbiter or his assistant and, if possible, by electronic means.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.3.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The player may at any time, when it is his move, ask the arbiter or his assistant to show him the scoresheet. This may be requested a maximum of five times in a game. More requests shall be considered as a distraction of the opponent.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Otherwise, play shall be governed by the Rapid chess Laws as in Article A.2 and A.4.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    B.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The regulations of an event shall specify whether Article B.3 or Article B.4 shall apply for the entire event.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Appendix C. Algebraic notation**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    FIDE recognises for its own tournaments and matches only one system of notation, the Algebraic System, and recommends the use of this uniform chess notation also for chess literature and periodicals. Scoresheets using a notation system other than algebraic may not be used as evidence in cases where normally the scoresheet of a player is used for that purpose. An arbiter who observes that a player is using a notation system other than the algebraic should warn the player of this requirement.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Description of the Algebraic System**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    In this description, &lsquo;piece&rsquo; means a piece other than a pawn.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Each piece is indicated by an abbreviation. In the English language it is the first letter, a capital letter, of its name. Example: K=king, Q=queen, R=rook, B=bishop, N=knight. (N is used for a knight, in order to avoid ambiguity.)<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    For the abbreviation of the name of the pieces, each player is free to use the name which is commonly used in his country. Examples: F = fou (French for bishop), L = loper (Dutch for bishop). In printed periodicals, the use of figurines is recommended.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Pawns are not indicated by their first letter, but are recognised by the absence of such a letter. Examples: the moves are written e5, d4, a5, not pe5, Pd4, pa5.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The eight files (from left to right for White and from right to left for Black) are indicated by the small letters, a, b, c, d, e, f, g and h, respectively.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The eight ranks (from bottom to top for White and from top to bottom for Black) are numbered 1, 2, 3, 4, 5, 6, 7, 8, respectively. Consequently, in the initial position the white pieces and pawns are placed on the first and second ranks; the black pieces and pawns on the eighth and seventh ranks.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.7<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    As a consequence of the previous rules, each of the sixty-four squares is invariably indicated by a unique combination of a letter and a number.<br/>    <br/>    <br/>    <br/>    <br/>    ![Description: A description...](/FIDE/handbook_images/field_number.png)<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    C.8<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Each move of a piece is indicated by the abbreviation of the name of the piece in question and the square of arrival. There is no need for a hyphen between name and square. Examples: Be5, Nf3, Rd1. In the case of pawns, only the square of arrival is indicated. Examples: e5, d4, a5.  <br/>     A longer form containing the square of departure is acceptable. Rules for play with blind and visually disabled players**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The organiser, after consulting the arbiter, shall have the power to adapt the following rules according to local circumstances. In competitive chess between sighted and visually disabled (legally blind) players either player may demand the use of two boards, the sighted player using a normal board, the visually disabled player using one specially constructed. Thisboard must meet the following requirements:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    measure at least 20 cm by 20 cm,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.1.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    have the black squares slightly raised,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.1.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    have a securing aperture in each square,<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.1.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The requirements for the pieces are:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.1.4.1<br/>    <br/>    <br/>    <br/>  <br/>    <br/>    <br/>    <br/>    <br/>    D.1.4.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    all are of Staunton design, the black pieces being specially marked.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/> <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The moves shall be announced clearly, repeated by the opponent and executed on his chessboard. When promoting a pawn, the player must announce which piece is chosen. To make the announcement as clear as possible, the use of the following names is suggested instead of the corresponding letters:<br/>    <br/>    <br/>    <br/>    <br/>    A - Anna   <br/>     B - Bella   <br/>     C - Cesar   <br/>     D - David   <br/>     E - Eva   <br/>     F - Felix   <br/>     G - Gustav   <br/>     H - Hector<br/>    <br/>    <br/>    <br/>    <br/>    Unless the arbiter decides otherwise, ranks from White to Black shall be given the German numbers<br/>    <br/>    <br/>    <br/>    <br/>    1 - eins   <br/>     2 - zwei   <br/>     3 - drei   <br/>     4 - vier   <br/>     5 - fuenf   <br/>     6 - sechs   <br/>     7 - sieben   <br/>     8 - acht<br/>    <br/>    <br/>    <br/>    <br/>    Castling is announced &ldquo;Lange Rochade&rdquo; (German for long castling) and &ldquo;Kurze Rochade&rdquo; (German for short castling).<br/>    <br/>    <br/>    <br/>    <br/>    The pieces bear the names: Koenig, Dame, Turm, Laeufer, Springer, Bauer.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    D.2.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    On the visually disabled player&apos;s board a piece shall be considered &lsquo;touched&rsquo; <br/>    <br/>    <br/>    <br/>    is illegal.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    I.9<br/>    <br/>    <br/>   <br/>    I.9.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the player having to reply to the sealed move is present, the envelope is opened, the sealed move is   <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the player having to reply to the sealed move is not present, his clock shall be started; on his arrival, he may stop his clock and summon the The starting time shall be announced in advance.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Guidelines II. Chess960 Rules**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Before a Chess960 game a starting position is randomly set up, subject to certain rules. After this, the game is played in the same way as regular chess. In particular, pieces and pawns have their normal moves, and each player&apos;s objective is to checkmate the opponent&apos;s king.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.2<br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Starting-position requirements**<br/>    <br/>    <br/>    <br/>    <br/>    The starting position for Chess960 must meet certain rules. White pawns are placed on the second rank as in regular chess. All remaining white pieces are placed randomly on the first rank, but with the following restrictions:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.2.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the king is placed somewhere between the two rooks, and<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.2.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the bishops are placed on opposite-coloured squares, and<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.2.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the black pieces are placed opposite the white pieces.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The starting position can be generated before the game either by a computer program or using dice, coin, cards, etc.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>      **Chess960 castling rules**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Chess960 allows each player to castle once per game, a move by potentially both the king and rook in a single move. However, a few interpretations of regular chess rules are needed for castling, because the regular rules presume initial locations of the rook and king that are often not applicable in Chess960.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    How to castle. In Chess960, depending on the pre-castling position of the castling king and rook, the castling manoeuvre is performed by one of these four methods:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    double-move castling: by making a move with the king and a move with the rook, or<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    transposition castling: by transposing the position of the king and the rook, or<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    king-move-only castling: by making only a move with the king, or<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    rook-move-only castling: by making only a move with the rook.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Recommendations:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.5.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    When castling on a physical board with a human player, it is recommended that the king be moved outside the playing surface next to his final position, the rook then be moved from its starting position to its final position, and then the king be placed on his final square.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.5.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    After castling, the rook and king&apos;s final positions should be exactly the same positions as they would be in regular chess.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Clarification:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Thus, after c-side castling (notated as 0-0-0 and known as queen-side castling in orthodox chess), the king is on the c-square (c1 for white and c8 for black) and the rook is on the d-square (d1 for white and d8 for black). After g-side castling (notated as 0-0 and known as king-side castling in orthodox chess), the king is on the g-square (g1 for white and g8 for black) and the rook is on the f-square (f1 for white and f8 for black).<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.7<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    Notes<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.7.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    To avoid any misunderstanding, it may be useful to state &quot;I am about to castle&quot; before castling.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.7.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    In some starting positions, the king or rook (but not both) does not move during castling.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.7.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    In some starting positions, castling can take place as early as the first move.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.7.4<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    All the squares between the king&apos;s initial and final squares (including the final square) and all the squares between the rook&apos;s initial and final squares (including the final square) must be vacant except for the king and castling rook.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    II.3.2.7.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    In some starting positions, some squares can stay filled during castling that would have to be vacant in regular chess. For example, after c-side castling 0-0-0, it is possible to have a, b, and/or e still filled, and after g-side castling (0-0), it is possible to have e and/or h filled.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Guidelines III. Games without increment including Quickplay Finishes**<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A &lsquo;quickplay finish&rsquo; is the phase of a game when all the remaining moves must be completed in a finite time.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.2.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The Guidelines below concerning the final period of the game including Quickplay Finishes, shall only be used at an event if their use has been announced beforehand.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.2.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    These Guidelines shall apply only to standard chess and rapid chess games without increment and not to blitz games.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.3.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If both flags have fallen and it is impossible to establish which flag fell first then:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.3.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the game shall continue if this occurs in any period of the game except the last period.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.3.1.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    the game is drawn if this occurs in the period of a game in which all remaining moves must be completed.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.4<br/>    <br/>    <br/>    If the player having the move has less than two minutes left on his clock, he may request that an increment extra five seconds be introduced for both players. This constitutes the offer of a draw. If the offer refused, and the arbiter agrees to the request, the clocks shall then be set with the extra time; the opponent shall be awarded two extra minutes and the game shall continue.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.5<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If Article III.4 does not apply and the player having the move has less than two minutes left on his clock, he may claim a draw before his flag falls. He shall summon the arbiter and may stop the chessclock (see Article 6.12.2). He may claim on the basis that his opponent cannot win by normal means, and/or that his opponent has been making no effort to win by normal means:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.5.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the arbiter agrees that the opponent cannot win by normal means, or that the opponent has been making no effort to win the game by normal means, he shall declare the game drawn. Otherwise he shall postpone his decision or reject the claim.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.5.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the arbiter postpones his decision, the opponent may be awarded two extra minutes and the game shall continue, if possible, in the presence of an arbiter. The arbiter shall declare the final result later in the game or as soon as possible after the flag of either player has fallen. He shall declare the game drawn if he agrees that the opponent of the player whose flag has fallen cannot win by normal means, or that he was not making sufficient attempts to win by normal means.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.5.3<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    If the arbiter has rejected the claim, the opponent shall be awarded two extra minutes.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.6<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The following shall apply when the competition is not supervised by an arbiter:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.6.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    A player may claim a draw when he has less than two minutes left on his clock and before his flag falls. This concludes the game. He may claim on the basis:<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.6.1.1<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    that his opponent cannot win by normal means, and/or<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.6.1.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    that his opponent has been making no effort to win by normal means.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    In III.6.1.1 the player must write down the final position and his opponent must verify it.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    In III.6.1.2 the player must write down the final position and submit an up-to-date scoresheet. The opponent shall verify both the scoresheet and the final position.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    III.6.2<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The claim shall be referred to the designated arbiter.<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>     <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    **Glossary of terms in the Laws of Chess **<br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    <br/>    The number after the term refers to the first time it appears in the Laws.<br/>    <br/>    <br/>      <br/>    <br/>    **adjourn: **8.1.
